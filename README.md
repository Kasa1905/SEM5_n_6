## 🚀 Quick Start (Ubuntu)

### Prerequisites
```bash
sudo apt update
sudo apt install openjdk-11-jdk gcc git
```

### Clone and Run
```bash
git clone https://github.com/Kasa1905/SEM5.git
cd SEM5/SPOS/EXAM/Q1
javac Pass1.java
java pass1
```

### Run Any Question
```bash
cd SPOS/EXAM/Q[number]     # Replace [number] with question number
javac *.java
java [MainClass]           # Replace with main class name
```
